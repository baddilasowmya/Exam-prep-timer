# Exam-prep-timer
A lightweight aptitude and exam preparation (such as TCS, Wipro, Amazon, Google, and aptitude drills. ) timer with interval alerts, pause/resume, and a floating always-on-top UI. Perfect for practicing timed aptitude questions and competitive exam problems.

How to Run the Project

Follow these steps to run the Aptitude Timer on your system:

1️⃣ Install Python

Make sure Python 3 is installed.
Check by typing in terminal:

python --version

2️⃣ Install Required Sound Library (Optional)

The timer works even without this, but for better sound:

pip install simpleaudio numpy

If this doesn’t install, the program will still run using default system beep.

3️⃣ Download or Clone the Project

If using Git:

git clone https://github.com/yourusername/your-repo-name.git
cd your-repo-name


Or manually download the .py file.

4️⃣ Run the Timer

Open terminal inside the project folder and run:

python aptitude_timer.py

(Replace with your file name.)

5️⃣ Start Using the Timer

Enter Hours
Enter Minutes
Enter Interval Minutes

Click Start

Use Pause / Resume / Reset as needed
The window stays always on top
Alerts will play automatically
